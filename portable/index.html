RTV Portable

Take (most of) the joy of RTV with you: to specialized websites like Crunchyroll!

  • Install Violentmonkey (or equivalent userscript add-on) for Chrome or Firefox.
  • Install the RTV Portable Userscript.
  • Go to Crunchyroll (only website supported, currently) and go to a supported series.
  • Click the "TV" button in the top-right corner of the page.
  • Sync away!
